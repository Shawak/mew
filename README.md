### Install

First of all, install ruby 2.3.1 [Setup]( https://gorails.com/setup/ubuntu/14.04) (until `gem install bundler`)

Then:

    git clone https://github.com/Shawak/mew.git
    cd mew
    bundle install
    chmod 755 run.sh
    
And use `./run.sh` to start mew!
  
###Updating

    cd mew
    git pull
    bundle install
